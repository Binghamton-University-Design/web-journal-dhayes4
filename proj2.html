ARTS 486

Exquisite Corpse

This project was inspired by the pandemic. It involves abstract and realistic things and feelings composed with vector drawings. The rules of this exquisite corpse project was that there was to be some type of pattern involved. The colors used will be black, white, and grays. The ratio used was 2:1 and the partners could see the pieces during the process.

Our rules:

  • Inspired by the pandemic.
  • Can be realistic and/or abstract.
  • Must have a pattern involved.
  • 2:1 ratio.
  • Composed with vector drawings.
  • We can both see the full piece during the process.

Project was created by Dora Hayes and Rebecca Colao
